Windows Forms — інтерфейс програмування додатків (API), відповідальний за графічний інтерфейс користувача і є частиною Microsoft .NET Framework. Даний інтерфейс спрощує доступ до елементів інтерфейсу Microsoft Windows за допомогою створення обгортки для Win32 API в керованому коді.

Всередині .NET Framework, Windows Forms реалізується в межах простору імен System.Windows.Forms.
Основні елементи управління Windows Forms.
Button являє стандартну кнопку, яку користувач може натиснути для виконання дій.
ComboBox відображає дані в випадаючому полі зі списком.
DataGridView надає гнучку, расширяемую систему для відображення і редагування табличних даних.
Label відображає текст, який не може бути змінений користувачем.
RadioButton представляє набір з двох або більше взаємовиключних варіантів вибору для користувача.
